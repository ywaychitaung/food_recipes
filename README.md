## Getting Started

Mac: 

Install Homebrew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Install Python via Homebrew

```bash
brew install python
```

## Run

```bash
bash startserver.sh
```
